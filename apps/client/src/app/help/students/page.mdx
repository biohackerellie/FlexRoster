# Student User Guide

## The Dashboard

When you first login to FlexRoster, You will be taken to your dashboard.

![Student Dashboard](https://github.com/biohackerellie/images/blob/main/apps/flexroster/student4.png?raw=true)

From here, you'll be able to see the list of all teachers, request to be in their class for Flex, or send them a message. You can also see your current assigned classes, and the status of any requests you have made.

## Making a Request

With each row of teachers, you'll see a calendar and chat bubble icon.
![Request Icons](https://github.com/biohackerellie/images/blob/main/apps/flexroster/student2.png?raw=true)

<Callout type="info">
  To make a request, click the calendar icon. This will open a pop up window
  where you can select what date you would like to request to be in that
  teachers classroom. You will only be able to request days that the teacher has
  marked as available. Available days are marked white, while unavailable days
  are greyed out. You may only request one day at a time. ![Request Pop
  Up](https://github.com/biohackerellie/images/blob/main/apps/flexroster/student3.png?raw=true)
</Callout>
<Callout type="success">
  You can also click the chat bubble icon to send a message to the teacher.
</Callout>

## Viewing Requests

You can view the status of your requests by clicking the `My Requests` button. This will open a pop up window where you can see the requests you've made, as well as their status.
